## 🐛 如果遇到问题

### 登录失败？

1. **检查浏览器控制台：**
   - 按 F12 打开开发者工具
   - 查看 Console 标签是否有错误
   - 查看 Network 标签，检查请求是否成功

2. **检查 Vercel 日志：**
   - 在 Vercel 项目页面
   - 点击 "Logs" 标签
   - 查看是否有错误信息

3. **确认配置：**
   - Vercel 环境变量 `NEXTAUTH_URL` 是否正确
   - Google OAuth 重定向 URI 是否正确

### 图像生成失败？

1. **检查 API Key：**
   - 确认 `NANO_BANANA_API_KEY` 环境变量已配置
   - 确认 API Key 有效

2. **查看错误信息：**
   - 浏览器控制台
   - Vercel 日志
